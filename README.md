# FIFA-World-Cup-2026-player-analysis

I worked on FIFA player performance data.

## What I did
- Cleaned the data (fixed country names, removed duplicates)
- Added new features (overall score)
- Made two ML models to predict creativity

## Results
- Linear Regression: 56%
- Random Forest: 60%

Random Forest worked better.

## Files
- fifa_cleaning(1).ipynb
- ml(2).ipynb
- fifa_cleaned_with_features.csv
- model_comparison.csv

## How to run
Open in Colab, upload the CSV, run the code.
